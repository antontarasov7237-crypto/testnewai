# Poxel Clone — Low-Poly Shooter

Веб-игра в стиле **poxel.io**: low-poly 3D шутер с собственными моделями, рулетками-кейсами, магазином, экипировкой скинов и рабочим геймплеем против ботов.

Сделан на **Three.js** (подгружается с CDN через import maps) — без сборки, открывается в браузере как есть.

## Что внутри

- **Геймплей FPS от первого лица** — WASD-движение, прыжок, бег, прицел, отдача, перезарядка, хедшоты.
- **3 оружия**: Pistol, SMG, Rifle — каждое со своими характеристиками (урон, скорострельность, разброс, дальность, патроны).
- **Боты-противники** с патрулированием, преследованием, проверкой линии видимости, прицельной стрельбой по игроку. Волны постепенно растут в сложности.
- **Процедурно сгенерированная low-poly карта** с укрытиями, деревьями, стенами и сеткой.
- **Магазин**: покупка оружия и переключение скинов.
- **Кейсы-рулетки** в стиле CS:GO: 4 кейса с разными вероятностями, анимированная прокручивающаяся лента, 6 редкостей (Common → Mythic). Дубликаты автоматически конвертируются в монеты.
- **Инвентарь** скинов, экипировка прямо из инвентаря.
- **Локальное сохранение** прогресса (монеты, рекорд, инвентарь, экипировка).
- **HUD**: HP-бар, броня, патроны, счёт, монеты, kill-feed, индикатор попадания, эффект урона.

## Управление

| Клавиша | Действие |
| --- | --- |
| W A S D | Движение |
| Space | Прыжок |
| Shift | Бег |
| ЛКМ | Стрельба |
| R | Перезарядка |
| 1 / 2 / 3 | Смена оружия |
| Esc | Пауза / выход в меню |

## Запуск

Откройте `index.html` в современном браузере (поддержка ES modules + WebGL).

Для разработки удобно поднять локальный HTTP-сервер, чтобы Three.js загрузился по import maps:

```bash
# Любой из этих способов подойдёт
python3 -m http.server 8080
# или
npx serve .
```

Затем откройте `http://localhost:8080`.

## Структура

```
index.html        — разметка, оверлеи меню/магазина/кейсов
styles.css        — стили UI
js/main.js        — точка входа, навешивает события на меню
js/game.js        — главный игровой цикл, сцена, рендер
js/player.js      — движение игрока, коллизии, HP
js/weapons.js     — оружие, стрельба, перезарядка, анимация рук
js/enemies.js     — AI ботов
js/models.js      — процедурные low-poly модели (персонаж, оружие, карта)
js/data.js        — данные: оружие, скины, кейсы, редкости
js/ui.js          — UI: магазин, рулетка, инвентарь, HUD
js/storage.js     — localStorage-сейв
```

## Заметки по балансу

- Стартовые монеты: **250** — хватит на 2 простых кейса.
- За убийство бота: **+12..20** монет и **+100..** очков (× редкость волны).
- Хедшот = **×2 урона** и бонусные очки.
- Награда волной растёт с её номером (HP бота, скорострельность, шотдамага).
